# Getting and Cleaning Data Course Project - Code Book

This code book was required as part of project submission.  It's purpose is to\
summarizes the resulting data fields in the tidy_df_[datestamp].txt.  The date\
stamp refers to the date of creation of the file. 

**Lables: **

* Subject - ID of subject
* Activity - The activity performed when measurements were observed


### Activity Labels

|ID|Description|
|:--|:---------|
|1|WALKING|
|2|WALKING_UPSTAIRS|
|3|WALKING_DOWNSTAIRS|
|4|SITTING|
|5|STANDING|
|6|LAYING|
